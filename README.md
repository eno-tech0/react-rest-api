# Учебный проект с курса по javascript

Проект был создан при помощи [Create React App](https://github.com/facebook/create-react-app).
В качестве REST API сервиса использовался [An API of Ice And Fire](https://www.anapioficeandfire.com/api)

## Запуск проекта

Установка всех зависимостей:
### `npm init`
Запуск в режиме разработки
### `npm start`
Запуск в режиме продакшн
### `npm run build`

Используемые технологии:
* styled-components
* React.js
* REST API
